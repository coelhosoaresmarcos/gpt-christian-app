🛡️ Fluxo Christian

Consolidador de baixas e otimizações

App web para transformar o fluxo do GPT Christian em um processo operacional: recebe as planilhas, preserva os dados originais, associa otimizações por OS normalizada + CodBarra, gera validações e entrega um Excel consolidado.

1. Envie os arquivos

Período de análise
Datas inclusivas. Se ficar vazio, todas as datas e hospitais serão analisados.

2. Estrutura proposta

  1. Frontend estático: HTML, CSS e JavaScript puro.
  2. Processador Christian: regra de leitura, associação, baixa e validação no navegador.
  3. ExcelJS via CDN: leitura e geração de .xlsx com abas formatadas.
  4. Download local: nenhum arquivo é enviado para servidor nesta primeira versão.

3. Regras contempladas

  • CONTROLE DE MEDICAMENTOS mantém todos os registros originais.
  • OS normalizada usa os 7 primeiros dígitos numéricos.
  • Associação segura usa OS + CodBarra; lote não bloqueia.
  • Itens excluídos saem de BAIXAR/RELATORIO e entram em VALIDACAO.
  • RELATORIO mostra todos os itens baixáveis, com ou sem otimização.
  • Excel final sempre contém as 6 abas obrigatórias.
